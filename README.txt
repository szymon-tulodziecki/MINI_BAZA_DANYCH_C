# Mini Baza Danych Laptopów

## Opis
Mini Baza Danych Laptopów to aplikacja stworzona w języku C, która umożliwia zarządzanie informacjami o laptopach. Aplikacja pozwala na tworzenie, przeglądanie, sortowanie i modyfikowanie bazy danych laptopów, a także na zapis i odczyt danych z plików.

## Funkcjonalności
- **Dodawanie laptopów** do bazy danych
- **Usuwanie laptopów** z bazy
- **Modyfikacja informacji** o laptopach
- **Przeglądanie** istniejącej bazy danych
- **Sortowanie** laptopów według różnych kryteriów (cena, model, pojemność, producent)
- **Zapis i odczyt** bazy z plików

## Wymagania
- Kompilator C (np. GCC)
- CMake (opcjonalnie, jeśli używasz CMake do budowy projektu)
